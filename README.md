# caso-4
Se genera una matriz del tamaño dado, cada campo es un pixel de la imagen y sus valores son los colores en formato RBG. Cada espacio de la matriz se le será asignado el valor del blanco por medio del metodo fill, luego de manera aleatoria se crean segmentos de líneas en escalas de grises sobrepuestos (aunque son aleatorias se controlan , esto se logra utilzando parametros de aparicion y posibilidad) y luego se generan gradientes a partir de los puntos más oscuros de la matriz(de manera aleatoria/controlada) .
